This chapter presents the fault injection component of the \ac{Fatori-V} layer. It explains how a campaign is configured, how injection targets are represented, how Area and Time Profiles define where and when faults are injected, and how the tool executes configuration memory and register injections through the supported backends. The chapter also describes the \ac{ACME} implementations used to generate \ac{SEM} addresses for the Basys3 and XCKU040 boards, and all developed \ac{FI} hardware.
