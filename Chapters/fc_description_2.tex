This chapter establishes the background required to understand the contributions presented in this thesis. It introduces the architecture of \ac{SRAM}-based \acp{FPGA} and the radiation effects that motivate \ac{FT} and \ac{FI}, surveys \ac{FI} techniques and the specific tools used in this work, describes the hardware \acp{FTM} referenced throughout the thesis, positions \ac{Fatori-V} relative to existing \ac{FT}/\ac{FI} platforms, and surveys open-source \ac{RISC-V} \ac{SoC} candidate architectures, selecting an initial baseline that guided the first \ac{Fatori-V} implementation attempt.
