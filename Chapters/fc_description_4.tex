This chapter introduces the EL2 platform, initially selected as \ac{Fatori-V}'s hardware backbone, by describing its core complex, the VeeRwolfX \ac{SoC} used in the RVfpgaEL2 Basys3 flow, and the available tools and execution workflows. It then presents the planned \acp{FTM} integrations, and a report on every modification attempted. The chapter concludes with the reasons that motivated the transition to the Ibex-based approach in \cref{chap5:ibex}.
