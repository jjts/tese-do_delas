This chapter discusses how \ac{Fatori-V} addresses the gap identified in \Cref{chap1:motivation} and summarizes the thesis’s main contributions and key findings, including core/platform integration work and the implemented fault-tolerance support and fault injection system. It then closes with a short set of future work directions, such as extending the set of \acp{FTM}, \ac{FI} models, an even higher layer for \ac{Fatori-V}, and overall performance refinements.
