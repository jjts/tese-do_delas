This chapter describes the developed Py2HWSW layer of \ac{Fatori-V}, an expansion to the IOb-\ac{SoC}-Ibex Setup Directory described in \cref{chap5:iob-soc-ibex}. It contains most of the hardware implemented in this work, including the configuration interface, the Ibex modifications required by \ac{Fatori-V}, the added \acp{FTM}, and the hardware used to monitor \ac{FT} experiments. It also includes the firmware and host-side console path used to read the monitoring \acp{CSR}, send the measured values through \ac{UART}, and create the measured-data file consumed by the upper layer.
