This chapter describes the hardware component of the Py2HWSW layer of \ac{Fatori-V}, built on top of the IOb-\ac{SoC}-Ibex baseline described in \Cref{chap5:iob-soc-ibex}. It presents the configuration interface and register-wrapping infrastructure, the M-of-N Redundancy and Self-Testing \acp{FTM} added to Ibex, and the hardware used to monitor and control \ac{FT} experiments.
