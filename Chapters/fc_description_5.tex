This chapter presents the IOb-SoC-Ibex baseline selected for \ac{Fatori-V}. It introduces the Ibex core, its \ac{CSR}-based performance monitoring interface, and its native \ac{FT} mechanisms. It then describes the integration work that produced IOb-SoC-Ibex: the Py2HWSW build infrastructure, the three-layer repository structure, and the \ac{AXI} bridge developed to connect the Ibex memory interface to the \ac{SoC} bus.
